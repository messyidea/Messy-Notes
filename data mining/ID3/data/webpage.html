数据集是从UCI找的，名称叫做“IndianLiver Patient Dataset”，这个数据集由三个印度的教授收集自印度安得拉邦的东北部，包含了416个肝癌病人和167个非肝癌病人共计583个肝病患者的病历数据记录。583个病人中包含441名男性病人和142名女性病人。完整的数据集包含以下内容：
X1 - age                                 年龄
X2 - gender                            性别
X3 - total Bilirubin                总胆红素
X4 - direct Bilirubin              直接型胆红素
X5 - Alkphos                         碱性磷酸酶
X6 - SGPT                                      谷丙转氨酶
X7 - SGOT                                     谷草转氨酶
X8 - total proteins                总蛋白
X9 - albumin                          白蛋白
X10 - A/G ratio                      白/球蛋白比
type - 类别                           1表示肝癌病人，2表示非肝癌病人
由于X10有缺失，剔除这个变量。

马氏距离判别法判别结果如下：训练样本判别率74.26%，测试样本判别率72%，训练样本和测试样本的判别率比较均衡。
#距离判别法
source('dist.ma.r')        #调用距离判别函数
trng.dst = dist.ma(ilpd.trng[, -10], ilpd.trng$type); trng.dst


table(trng.dst[, 4], trng.dst[, 1])
trng.wr = which(trng.dst[, 1] != trng.dst[, 4])
length(trng.wr)
length(ilpd.trng$type)
r11 = length(trng.wr)/length(ilpd.trng$type); 1 - r11


test.dst = dist.ma(ilpd.trng[, -10], ilpd.trng$type, ilpd.test[, -10])
test.dst
table(test.dst$newG, ilpd.test[, 10])
test.wr = which(test.dst$newG != ilpd.test[, 10])
length(test.wr)
length(ilpd.test$type)
r12 = length(test.wr)/length(ilpd.test$type); 1 - r12


#决策树
library(rpart)
trng.rp = rpart(type ~ ., data = ilpd.trng, method = 'class')
library(rpart.plot)
rpart.plot(trng.rp, col = 'blue', main = 'Decision Tree')

决策树算法分类的结果如下：训练样本判别率83.09%，测试样本判别率66.86%，训练样本的效果比较好，测试样本的判别率不算高。

#神经网络
library(nnet)
trng.nnet = nnet(type ~ ., data = ilpd.trng, size = 20, rang = 2e-4,
        decay = 5e-4, maxit = 2000, trace = F, skip = T, Hess = T, Linout = T)


trng.nnet.pre = predict(trng.nnet, ilpd.trng, type = 'class')
tanet.trng = table(trng.nnet.pre, ilpd.trng$type); tanet.trng


test.nnet.pre = predict(trng.nnet, ilpd.test, type = 'class')
tanet.test = table(test.nnet.pre, ilpd.test$type); tanet.test


length.trng.wr7 = tanet.trng[1, 2] + tanet.trng[2, 1]
r31 = length.trng.wr7/length(ilpd.trng$type); 1 - r31
length.test.wr7 = tanet.test[1, 2] + tanet.test[2, 1]
r32 = length.test.wr7/length(ilpd.test$type); 1 - r32


神经网络方法分类结果如下：训练样本判别率97.55%，测试样本判别率68%。感觉神经网络还是存在过拟合的问题，训练样本的判别率过高了，
调整了好多次参数，测试样本的判别率最高也就达到70%。在这个问题上，神经网络方法的效果不比决策树好。
总的来说，可能选择的样本两个类别的差异不是太明显，致使决策树和神经网络的效果都不理想，反倒是距离判别法更加稳健。
